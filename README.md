📊Project Title: Client Support Operations Analytics

📌 Project Overview: In my role as a Client Support Executive, I manage operations for various niche magazines like Finance, Pharma, and Manufacturing. Handling daily client responses and feedback cycles via manual emails was time-consuming and prone to errors. To solve this, I developed an automated analytics solution that replaces manual tracking with a dynamic Power BI dashboard.

🛠️ Tech Stack: 

Microsoft Excel: Data cleaning, Power Query, and structured formatting.Power BI: Data Modeling, DAX (Calculated Measures), and Interactive Dashboards.

🚀 The Solution & Implementation:

Automated Tracking: Converted manual email logs into a structured dataset for real-time reporting.
KPI Development: Built metrics like Response Rate %, Interview Counts, and Approval Status.
Interactive Visuals: Used Gauge Charts for target tracking and Slicers for magazine-wise deep dives.

💡 Key Insights & Impact:

📈 Performance Boost: Identified that certain magazines exceeded response targets by 124%, allowing for better resource reallocation.
⏱️ Efficiency Gain: Automated the workflow, saving 5–10 hours per week of manual tracking.
🔍 Transparency: Real-time visibility into "Draft Pending" vs "PDF Approved" status led to faster client follow-ups and better coordination.
