📊 Project 1: Operational Performance & Client Response Dashboard

Project Overview:In my daily role as a Client Support Executive, I manage operations for multiple magazines (Finance, Pharma, Manufacturing, etc.). Manually tracking daily responses, interview status, and feedback cycles via emails was time-consuming and prone to errors. I built this project to automate the reporting process and gain data-driven insights into team performance.

Tech Stack Used:Microsoft Excel: Data cleaning, initial processing, and pivot table analysis.Power BI: Data modeling (Relationship management), DAX (Calculated measures), and interactive visualization.

What I Implemented 

Automated Tracking: Converted manual email reports into a structured dataset.KPI Development: Built key metrics like Response Rate %, Interview Counts, and Approval Status (Pending vs. Approved).Interactive Visuals:Gauge Chart: To track 'Actual Responses' against 'Expected Targets'.

Clustered Bar Chart: For magazine-wise performance comparison.

Slicers: Added interactivity to filter data by specific Magazine names.

Data Modeling: Established relationships between multiple tables to ensure seamless data flow across the dashboard.Key Insights & Impact:Bottleneck Identification: Identified specific magazines (e.g., Industry Outlook) that consistently exceeded response targets by 124%, while others needed resource reallocation.

Efficiency: Automated the reporting workflow, saving approximately 5-10 hours of manual tracking per week.Transparency: Real-time visibility into "Draft Pending" vs "PDF Approved" status helped in faster client follow-ups.
