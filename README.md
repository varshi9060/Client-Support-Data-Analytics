📊 Client Support Operations & Performance Analytics :

🎯 Project OverviewIn my daily role as a Client Support Executive, I manage operations for multiple magazines (Finance, Pharma, Manufacturing, etc.). Tracking daily responses and feedback was a manual, time-consuming process. I built this project to automate reporting and get better insights into our team's performance.

🛠️ Tech Stack & Skill
Excel:Data cleaning and initial processing.
MySQL: Database management and advanced querying.
Power BI: Interactive dashboarding and KPI visualization.

🚀 Project 1: Operational Performance Dashboard (Power BI)
I transformed scattered manual reports into an interactive dashboard to track real-time status.
What I did:Built a dashboard tracking Response Rates, Interview counts, and Approval statuses.
Key Insight: Identified that the 'Industry Outlook' magazine exceeded its response target by 124%.
Result: Automated the reporting process, saving hours of manual tracking every week.
